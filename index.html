About this Website
This website was created entirely by me, and I created it due to many players having lots of trouble figuring out how to be an officer and having many questions in the Department. This websites purpose is to allow for new AND experienced officers to have extra resources when going on duty as any rank in the Monoford or Pine County Police Department. It contains a calculator, which can help you determine the range of minutes you should put a suspect in prison for due to the amount of charges they have (as well as helping you with tickets). It also contains a page about the Chain of Command in the Monoford County Police Department, and a page with all useful police 10-Codes.
